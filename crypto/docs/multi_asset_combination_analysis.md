最后更新：2026-05-20

## 背景

对 BTC / ETH / SOL / DOGE 四个标的，各自的三种策略版本（EMA、V2、V3）进行全面回测分析，评估单标的内部融合、跨标的两两融合、三标的组合、全组合的收益与回撤表现。

数据来源：TradingView 策略导出 Excel，OKX 永续合约，初始资金各 $50,000。  
数据范围：约 2019-12 ~ 2026-05。  
DOGE 去掉 EMA 策略（最大回撤 44.28%，夏普 0.065，不可用）。

分析脚本：`crypto/analysis/run_analysis.py`（一键入口，含所有分析模块）  
图表目录：`crypto/analysis/charts_2026-05-18/`

## 核心结论

### 1. ETH 是最强单标的

ETH V2 夏普 0.425、V3 盈利因子 1.806，两个维度均领先。ETH 三策略融合后回撤 8.49%，仍在可接受范围。

### 2. SOL 夏普最高，但回撤略大

SOL V2 夏普 0.450 是全场最高，但单策略回撤 8.26%，三策略融合后 8.59%（EMA 拖累）。V2+V3 组合回撤 8.12% 是 SOL 内部最优。

### 3. BTC 偏弱，但分散化价值高

BTC 三策略夏普均在 0.3 以下，年化最低。但 BTC 与 ETH/SOL 的相关性较低，加入组合后对回撤的改善效果显著（ETH+SOL 4.85% → BTC+ETH+SOL 4.87%，几乎不变，说明 BTC 没有拖累）。

### 4. DOGE EMA 必须排除

DOGE EMA 最大回撤 44.28%，夏普 0.065，不可用。DOGE V3 表现意外不错（夏普 0.323，回撤 9.32%），是 DOGE 唯一值得配置的策略。加入 DOGE 后任何组合回撤都会上升。

### 5. 最优两标的组合：ETH+SOL（回撤 4.85%）

ETH 和 SOL 都是强标的，两者融合后回撤最低，且收益不损失。

### 6. 最优三标的组合：BTC+ETH+SOL（回撤 4.87%）

加入 BTC 后回撤几乎不变（4.85% → 4.87%），但增加了 BTC 的收益贡献和分散化。

### 7. 策略层面分散优于标的层面

全11策略等权（5.30%）略优于全4标的等权（5.61%），说明在策略层面做等权分散比在标的层面更有效。

## 配置建议

| 标的 | 推荐策略 | 理由 |
|------|---------|------|
| ETH  | V2 + V3 等权 | 夏普最高，盈利因子最强，两策略相关性低 |
| SOL  | V2 单独 或 V2+V3 | V2 夏普全场最高；加 V3 可小幅降低回撤 |
| BTC  | EMA+V2+V3 等权 | 单标的弱但分散化价值高，三策略融合回撤仅 6.21% |
| DOGE | V3 轻仓 或 不配 | 加入后拖累整体回撤，如配则仅用 V3 |

**最优全组合方案**：ETH(V2+V3) + SOL(V2) + BTC(EMA+V2+V3)，不含 DOGE，预期最大回撤约 4.9%。

## 2026-05-20 更新：ETH 和 SOL 新版本分析

### ETH：V4（205m）替代 V3（3h）

ETH 新增 V4（205m 时间框架），与 V3 相关性 0.934，同时跑无分散化价值。V4 各项指标全面优于 V3：

| 指标 | V3 (3h) | V4 (205m) |
|------|---------|-----------|
| 年化% | 14.97 | **15.40** |
| 最大回撤% | 6.56 | **3.77** |
| Sortino | 1.223 | **1.719** |

**ETH 更新为：V2 + V4 等权**（原 V2+V3）。

### SOL：发现 V3（205m）零相关策略

SOL 新增五版本分析（V1/V2/V3_205m/V4_205m/V5_3h），发现 V3（205m）与其他所有策略相关性接近零（0.002~0.163）：

| 组合 | 年化% | 最大回撤% | Sortino |
|------|------|---------|---------|
| **V2+V3_205m** | **30.78** | **4.22** | **7.50** |
| V3_205m 单独 | 34.51 | 5.63 | 5.74 |
| V2 单独 | 25.85 | 6.71 | 3.39 |

**SOL 更新为：V2 + V3（205m）等权**（原 V2 单独）。

注意：SOL 的 V3 是 205m，不是 3h，与 ETH 版本命名不对应。

### 更新后的配置建议

| 标的 | 推荐策略 | 变化 |
|------|---------|------|
| ETH  | **V2 + V4（205m）等权** | V3 → V4 |
| SOL  | **V2 + V3（205m）等权** | V2 单独 → V2+V3 |
| BTC  | EMA+V2+V3 等权 | 不变 |
| DOGE | V3 轻仓 或 不配 | 不变 |

## 与上次分析的差异

上次分析（2026-05-13，仅 ETH 和 SOL）结论：ETH 用 v3_3h+v2，SOL 用 v2 单独。  
本次分析新增 BTC 和 DOGE，结论一致：ETH 和 SOL 是核心标的，BTC 有分散化价值，DOGE EMA 不可用。
