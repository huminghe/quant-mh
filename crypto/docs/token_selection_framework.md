# 趋势跟踪标的筛选框架

最后更新：2026-05-13

## 背景

通过对 JUP（表现优秀）、ARB/LDO/LINK（深度回撤）的对比分析，提炼出一套判断代币是否适合趋势跟踪策略的框架。核心结论：**决定趋势跟踪能否盈利的，不是代币涨跌，而是价格结构是否允许趋势延续。**

---

## 正向信号（趋势友好）

| 信号 | 说明 |
|------|------|
| 内部人分配 < 15% | 或主要解锁已完成/暂停 |
| 真实协议收入回购机制 | 收入直接流向代币持有者或用于回购 |
| 上市 < 2 年 | 仍在价格发现阶段，新信息驱动单向价格移动 |
| 持有者分散 | 数十万以上独立地址，散户动量追逐行为强化趋势 |
| 生态系统处于上升叙事周期 | 定向资本流入具有持续性和方向性 |
| 链上流动性深度 > $100 万 | 大单不会造成过度滑点，趋势可以平滑发展 |
| 质押锁仓比例高 | 减少自由流通供应，价格对买需更敏感 |

## 负向信号（趋势杀手）

| 信号 | 说明 |
|------|------|
| VC/团队持有 > 30% 且仍在线性解锁 | 创造可预测的月度天花板，市场提前 30 天定价卖出压力 |
| 纯治理代币，无收益分配，无质押收益 | 无价格下限的基本面支撑，无持有激励 |
| 鲸鱼集中度 > 80% | 少数大户可随意操控价格，趋势信号不可靠 |
| 机构钱包定期向 Coinbase Prime 转账 | 链上可追踪的系统性卖出，可预测的顶部压力 |
| 链上流动性深度 < $50 万 | 中等规模买卖造成剧烈波动，产生噪音而非趋势 |
| 协议收入极低或无收入 | 估值完全依赖叙事，无基本面锚点 |
| 生态系统持续萎缩 | 用户和开发者流失，趋势信号持续性差 |

---

## 趋势形成的底层机制

**为什么有些代币产生干净趋势，有些只有噪音：**

1. **供应天花板效应**：月度解锁创造可预测的顶部压力。市场参与者提前 30 天定价这种压力，产生 30 天周期的均值回归节奏，完全破坏趋势延续性。

2. **价格发现阶段**：新上市代币（< 2 年）市场对真实价值存在广泛分歧，新信息到来时价格会单向移动较长时间，因为没有"历史均值"可以回归。

3. **回购的不对称效应**：协议收入回购在价格下跌时提供支撑，在价格上涨时不产生额外卖出压力，形成不对称的价格动态，有利于上涨趋势的形成和维持。

4. **散户动量文化**：持有者分散（数百万地址）+ 散户正反馈交易行为，是趋势自我强化的关键驱动力。鲸鱼主导的市场（如 LDO 88.5% 鲸鱼集中度）倾向于均值回归。

## 回撤计算口径

用固定分母（每个标的初始投入资金，通常 20,000 USDT）计算回撤，而非用峰值净值作为分母。这样能真实反映实际资金损失幅度，避免因初始资金基数不同导致的失真。

**公式：** 回撤% = (当前累计P&L - 峰值累计P&L) / 初始投入资金

## 停止运行的决策原则

**核心问题：** 如果今天是第一次看到这个标的，知道它的代币经济模型，还会选择在上面跑趋势跟踪策略吗？

如果答案是否，继续运行的唯一理由就是沉没成本——已经亏的钱不会因为继续运行而回来，但未来的亏损是可以停止的。

**"等回本再停"是自我矛盾的逻辑：** 如果策略在这个标的上的期望值是负的，等待只会让亏损更多。
