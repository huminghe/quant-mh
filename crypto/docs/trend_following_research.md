# 趋势跟踪策略研究笔记

## 双均线策略的本质

双均线不产生 alpha，是**趋势存在时的收割工具**。alpha 来自市场的趋势性，不来自均线工具本身。换参数、换均线类型只是边际改善，无法解决根本问题——震荡市必然亏损。

学术研究结论：Brock 等 1992 年研究在样本内（1897-1986）有效，但样本外（1987-2011）完全失效，是技术分析领域最典型的数据窥探案例。

---

## 失效条件

**市场结构层面：**
- 震荡横盘：均线频繁交叉，连续小额亏损（whipsaw）
- 政策驱动突然反转：2025 年 4 月关税反复横跳，SG 趋势指数一周损失 6 个百分点
- 趋势换向频率提高：2010 年后平均每年换向 3.0 次（vs 1990-2009 年的 2.3 次），结构性变化

**跨资产有效性对比：**

| 资产类别 | 有效性 | 原因 |
|---|---|---|
| 商品期货 | 强 | 供需驱动持续趋势 |
| 外汇 | 中等 | 宏观政策驱动，但央行干预截断趋势 |
| 股票指数 | 弱（近30年） | 牛市中持续跑输买入持有 |
| 个股 | 很弱 | 噪声大，均值回归主导 |
| 加密货币 | 中等偏强 | 明显牛熊周期，但 2021 年后恶化 |

---

## 失效判断的量化指标

| 指标 | 正常范围 | 触发审查 |
|---|---|---|
| 滚动 12 月 Sharpe | > 0 | 连续 2-3 个月 < 0 |
| 信号胜率 | 35-45% | 持续低于 30% 且盈亏比同时下降 |
| 连续亏损次数 | 5-7 次 | 10 次以上 |
| 市场整体 ADX 均值 | > 20 | 长期低于 20 |

**区分暂时回撤 vs 策略失效：**
- 暂时回撤：市场仍有趋势，只是当前调整期；ADX 仍在合理水平；回撤在历史分布内
- 策略失效：所有参数组合同时失效；跨资产类别同时失效；市场结构发生根本变化

## 改进方向（按优先级）

### 1. ATR 动态定仓（最重要）

CTA 行业标准做法，Winton 最早系统化使用：

```
仓位大小 = 风险金额 / (ATR × 乘数)
```

高波动时自动减仓，低波动时自动加仓，使每笔交易承担固定风险。不改变信号逻辑，只改变仓位大小，工程改动最小，对回撤控制效果显著。

### 2. ER 入场过滤（已验证有效，待集成）

见下方"入场过滤验证结论"章节。v1/v2 用 N=7/≥0.35/2H，Pine Script 代码已给出。

### 3. KAMA 自适应均线

Pine Script 无内置函数，需手动实现（代码见下方）。

**核心机制：效率比（ER）动态调整平滑系数**

```
ER = |当前价格 - N日前价格| / Σ|每日价格变化|
```

- ER 接近 1（强趋势）→ 均线快速跟随
- ER 接近 0（震荡）→ 均线几乎静止

BTC 回测：whipsaw 率从 38% 降到 14%，利润因子从 1.21 提升到 1.48。

**Pine Script 实现：**

```pine
//@version=5
indicator("KAMA", overlay=true)

length = input.int(10, "回望期")
fast   = input.int(2,  "快速EMA周期")
slow   = input.int(30, "慢速EMA周期")

fast_sc = 2.0 / (fast + 1)
slow_sc = 2.0 / (slow + 1)

direction  = math.abs(close - close[length])
volatility = math.sum(math.abs(close - close[1]), length)
er         = volatility != 0 ? direction / volatility : 0

sc = math.pow(er * (fast_sc - slow_sc) + slow_sc, 2)

var float kama = na
kama := na(kama[1]) ? close : kama[1] + sc * (close - kama[1])

plot(kama, color=color.blue, linewidth=2)
```

## CTA 行业历史周期

| 时期 | SG 趋势指数 | 原因 |
|---|---|---|
| 2009-2018 | 年化约 +0.4% | 量化宽松压制波动，失落十年 |
| 2022 | 全年约 +20% | 俄乌战争 + 加息周期，趋势明确 |
| 2025 H1 | -10.48% | 关税政策反复横跳，趋势频繁被截断 |

**核心规律：** 趋势策略在宏观政策反复横跳时系统性失败，无法通过参数优化规避，只能通过仓位管理和暂停机制控制损失。

## 制度过滤信号验证结论（2026-05-27）

### 验证范围

4个策略（BTC/ETH/SOL/DOGE_ema），4189笔实盘交易，8个市场指标，日线时间框架。

### 单笔交易层面：无效

| 指标 | 平均区分度 | 结论 |
|------|-----------|------|
| 相对MA200% | +2.0 | 最高，但量级在噪声范围内 |
| 年化波动率% | +1.5 | 无实用价值 |
| 21日动量% | +1.1 | 无实用价值 |
| ATR百分位 | +0.9 | 无实用价值 |
| Choppiness Index(14) | -0.4 | 无效 |
| ADX(14) | -0.2 | 无效 |
| BBW布林带宽 | -0.2 | 无效 |
| 收益率自相关lag5 | -0.01 | 完全无效 |

区分度 = 盈利交易均值 - 亏损交易均值，绝对值越大预测能力越强。

**根本原因：** 低胜率（31-37%）+高盈亏比策略，亏损是随机分布的正常状态，不集中于特定市场环境。单笔交易层面找不到规律是预期内的结果。

### 月度层面：有信号，但无法用于实时过滤

77个月数据，亏损月26个，盈利月51个：

| 指标 | 亏损月均值 | 盈利月均值 | 差值 |
|------|-----------|-----------|------|
| 月末相对MA200% | +2.9% | +16.0% | -13.1 |
| 月胜率% | 28.1% | 37.7% | -9.6 |
| ADX月均值 | 26.8 | 29.9 | -3.1 |
| CI月均值 | 50.7 | 49.3 | +1.4 |

CI分箱发现：CI月均值 > 55 时，盈利月比例仅20%（5个月中4个亏损）。ADX月均值 > 30 时，盈利月比例81%。

**局限性：** 月度指标是事后统计，月末才知道当月均值，无法用于月初决策。

### ADX 16维度验证

8H+日线 × 自身+BTC × 4策略 = 16张表，无一呈现单调趋势。BTC 8H ADX < 10 仅占全部交易的1.2%，即使有效也无实用价值。

### 结论

**不需要替换或增强现有暂停机制。** 现有 Stoch+BBWP+12%跌幅 的逻辑（大跌+高波动后横盘）已是合理方案，5次触发80%命中率，实盘验证过。制度过滤信号的研究到此为止。

## 入场过滤验证结论（2026-05-27）

### 验证范围

v1/v2/v3_205m/v3_3h 四个策略版本，BTC/ETH/SOL/DOGE，2019-2026，共约 13,767 笔交易。

### Choppiness Index (CI) 入场过滤

CI(10) 2H ≤50 对 v1/v2/v3_3h 有效，v3_205m 效果弱。

| 版本 | 基准 PnL | 过滤后提升 |
|------|---------|-----------|
| v1 (479m) | +139,687 | +80,099 (+57%) |
| v2 | +85,192 | +51,413 (+60%) |
| v3_3h SOL | — | +30,019 |
| v3_205m | — | 弱（+6%） |

- 2H 是跨版本最稳定的时间框架，30m 全部负向
- 被过滤交易胜率 19-27%，保留交易胜率 39-47%，规律一致
- **2025 年 CI 失效**（v1 合计 -2,732），是主要缺陷

### Efficiency Ratio (ER) 入场过滤

ER 直接测量价格运动效率，与 CI 同维度但线性、反应更快。**全面优于 CI**，尤其在 2025 年 CI 失效时 ER 仍有效。

**N 参数与策略时间框架强相关：**
- v1/v2（479m，2H 过滤）：**N=7 最优**（v1 最高 +97%）
- v3_205m/v3_3h（更短周期，1H 过滤）：**N=10 最优**，N=5/7 反而全部负向

**统一参数方案（N=10, 阈值 ≥0.25）：**

| 版本 | 提升 | CI 对比 |
|------|------|---------|
| v1 (2H) | +67% | CI +43% |
| v2 (2H) | +38% | CI +11% |
| v3_205m (1H) | +15% | CI +6% |
| v3_3h (1H) | +46% | CI +22% |
| **全版本合计** | **+34.3%** | CI +16.7% |

**年度稳定性（v1，ER≥0.3 vs CI≤50，2H）：**

| 年份 | ER≥0.3 提升 | CI≤50 提升 |
|------|------------|-----------|
| 2020 | +17,510 | +13,364 |
| 2021 | +26,247 | +23,033 |
| 2022 | +22,048 | +18,395 |
| 2023 | +13,843 | +11,572 |
| 2024 | +21,724 | +9,166 |
| 2025 | +21,163 | **-2,732** |
| 2026 | +8,986 | +7,110 |

**结论：用 ER 替换 CI，不叠加。** CI+ER 组合不如 ER 单独（CI 过滤掉了部分 ER 认为值得做的好交易）。

### ACF（自相关系数）入场过滤

**完全无效，已排除。**

1H/2H/4H/8H/1D 全部时间框架，lag=1/3/5，所有阈值均大幅负向（-40% 到 -107%）。ER+ACF 组合也比 ER 单独更差。

根本原因：ACF 测量历史收益率的统计特性（过去 20 根 K 线的自相关），与趋势策略盈利交易的发生时刻没有统计关联。ER 有效是因为直接测量当前 K 线的运动效率，两者信号逻辑不在同一尺度。

### CI 出场过滤

**无效，已排除。**

所有时间框架（2h/4h/8h/1d）均无效。4h CI>61.8 有一定正向信号（+24,628），但 83% 由 DOGE 单一标的贡献，跨标的不一致。触发率过高（84-88%）截断大盈利是根本问题。CI 适合入场过滤，不适合出场过滤。

### ER 参数细化（2026-05-29）

v1/v2 专项对比：N=7 vs N=10，固定阈值 vs 百分位，`[1]` vs 不加 `[1]`。

**`[1]` vs 不加 `[1]`（关键发现）：**
- v1/v2 是 479m K线，入场时刻在 2H K线收盘前 1-3 分钟，当前 2H K线已接近收盘
- N=7 不加 `[1]` 比加 `[1]` 好 1-9%；N=10 不加 `[1]` 比加 `[1]` 好 20-38%
- 结论：**不加 `[1]`**，用当前 K 线的 ER，信息更新鲜，实际无前瞻偏差

**v1/v2 最终参数：N=7，阈值 ≥0.35，不加 `[1]`，2H，lookahead_on**

| 参数 | v1+v2 合计提升 | 保留率 |
|------|------|------|
| N=7 当前 ≥0.35 | **+59.5%** | 55-58% |
| N=10 当前 ≥0.30 | +61.2% | 52-53% |
| N=7 [1] ≥0.35 | +60.6% | 59% |

N=7 和 N=10 效果接近（差 1-2%），N=7 在高阈值（≥0.40）时更优（+69% vs +56.5%）。

**Pine Script（v1/v2）：**

```pine
f_er(length) =>
    math.abs(close - close[length]) / math.sum(math.abs(close - close[1]), length)

er_2h = request.security(syminfo.tickerid, "120", f_er(7), lookahead=barmerge.lookahead_on)
er_filter = na(er_2h) or er_2h >= 0.35

// 入场条件加：and er_filter
```

### ER 多空拆分（2026-05-29）

空头的 ER 过滤效果始终优于多头（约 10-20 个百分点），在所有阈值下一致：

| 阈值 | 全部 | 多头 | 空头 |
|------|------|------|------|
| ER≥0.35 | +59.5% | +54.5% | +65.7% |
| ER≥0.40 | +69.0% | +60.7% | +79.4% |

原因：加密货币下跌趋势往往更"干净"（恐慌性抛售方向效率高），ER 对空头信号的区分度更强。多头也是正向的，不需要关掉多头过滤。

### 百分位 ER vs 固定阈值（2026-05-29）

百分位 ER（滚动 100 期 percentrank）不优于固定阈值：

| 方案 | v1+v2 合计提升 |
|------|------|
| 固定阈值最优（ER≥0.40） | +69.0% |
| 百分位最优（ER_pct≥60） | +64.2% |

结论：维持固定阈值，不换百分位。原因：4 个标的的跨标的不一致问题不明显，百分位引入额外窗口参数反而增加过拟合风险。

### Hurst 指数（DFA）入场过滤（2026-05-29）

**完全无效，已排除。**

全版本全阈值负向（最差 -175%）。加密货币 H 中位数 0.61，91% 时刻 H>0.5，过滤器几乎不过滤任何东西。ER+H 组合也不优于单独 ER。根本原因同 ACF：测量历史统计特性，与盈利时刻无关。

### ADX 入场过滤（2026-05-29）

**无效，已排除。**

ADX>20 全版本 -21%，ADX>25/30 更差（-54%~-81%）。ER+ADX 最优 +26.2%，低于单独 ER +34.3%。根本原因：趋势启动时 ADX 还在低位，过滤掉的恰好是趋势初期最赚钱的入场点。

### 成交量确认（2026-05-29）

**完全无效，已排除。**

所有时间框架（2H/4H/8H/1D）、所有倍数（1.2x~3.0x）全部大幅负向。ER+成交量最优 -22.6%，比基准还差。加密货币趋势跟踪不依赖放量突破，量大反而可能是顶部/底部反转信号。

### 资金费率入场过滤（2026-05-29）

**无效，已排除。**

四种逻辑（中性过滤/方向过滤/极端反向/排除同向）最优结果仅 +0.0%。关键背景：84% 的入场时刻资金费率为正，中位数恰好是 Binance 标准费率 0.01%，几乎没有区分度。文献中的"费率翻正=制度转换"是月度宏观信号，单笔层面无效。

### 入场过滤器系统性验证总结

| 指标 | 结论 | 最优提升 |
|------|------|------|
| ER（效率比） | **有效，最优方案** | v1/v2 +59.5%（N=7/≥0.35/2H） |
| CI（Choppiness Index） | 有效但被 ER 超越，2025 年失效 | +43% |
| ACF（自相关系数） | 无效 | -40%~-107% |
| Hurst 指数（DFA） | 无效 | 最差 -175% |
| ADX 入场过滤 | 无效 | -21%（ADX>20） |
| 成交量确认 | 无效 | -34%（量比≥1.2x） |
| 资金费率过滤 | 无效 | +0.0% |
| 百分位 ER | 不优于固定阈值 | +64.2% vs 固定 +69.0% |

**结论：ER 是唯一有效的入场过滤器。入场过滤器方向的系统性验证已完结。**
